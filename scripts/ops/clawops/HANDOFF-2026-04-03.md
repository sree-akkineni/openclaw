## 2) Channel/Number Reality (Important Constraint)

- Built-in OpenClaw WhatsApp channel is WhatsApp Web (`Baileys`) and requires linking from a real WhatsApp app account.
- Twilio numbers cannot be QR-scanned into this built-in channel.
- For digital-only phone access without WhatsApp app linking, use voice/SMS provider channels (for example `voice-call` with Twilio).
- If WhatsApp without QR is required, it needs a separate Twilio/Meta WhatsApp API adapter path (not the built-in Web channel flow).
